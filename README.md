# نظام تداول آلي احترافي (Professional Trading Bot)

نظام متكامل للتداول الآلي على شبكات EVM (Base، Arbitrum، Polygon) مع تحليل أمان متقدم، Oracle حي من Chainlink، تنبيهات Telegram، وإدارة مخاطر كاملة.

## المميزات الرئيسية
- 🚀 استماع لحظي لأزواج جديدة عبر WebSocket.
- 🔐 تحليل أمان التوكنات باستخدام GoPlus API (Honeypot، ضرائب، ملكية).
- 💧 حساب السيولة بالدولار باستخدام سعر ETH/USD الحي من Chainlink.
- 📊 إدارة مخاطر صارمة (حد أقصى للصفقات، خسائر متتالية، حد يومي).
- 💰 تنفيذ صفقات Buy/Sell مع انزلاق محدد وغاز متحكم به.
- 🎯 استراتيجية خروج متدرجة (Take Profit جزئي، Trailing Stop، Stop Loss).
- 📈 إدارة مراكز متعددة مع تحديثات مستمرة.
- 📱 تنبيهات فورية عبر Telegram لكل حدث مهم.
- 🧪 وضع Paper Trading للمحاكاة بدون مخاطرة.
- 🖥️ واجهة CLI للتحكم بالبوت (تشغيل، إيقاف، تغيير الشبكة، تفعيل Paper).
- 💾 تخزين جميع البيانات في SQLite.
- 🔄 إعادة اتصال تلقائي عند انقطاع WebSocket.

## متطلبات التشغيل
- Node.js 18+
- VPS (يوصى بـ 4GB RAM، 2 vCPU) قريب من مزود RPC
- محفظة بها رأس مال كافٍ (محفظة مخصصة للبوت)
- مفاتيح API: Alchemy/QuickNode (WebSocket)، GoPlus (اختياري)، Telegram Bot
- عنوان Oracle Chainlink (موجود في `.env.example` لكل شبكة)
